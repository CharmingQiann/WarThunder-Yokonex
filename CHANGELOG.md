# 更新日志

## 1.1.2 - 2026-07-23

- 强制插件标准输出和错误输出使用 UTF-8
- 解决 GameHub 按 UTF-8 读取 Windows 插件日志时出现的中文乱码

## 1.1.1 - 2026-07-23

- 删除插件侧 A/B 强度上限配置
- 连续输出改为读取 GameHub 中 commandId 对应的设备波形
- 以每台设备波形的 A/B 峰值作为连续输出上限
- 多设备分别使用自己的波形和强度
- 配置精简为游戏昵称一个字段
- 过载、速度、CAS、档位阈值和轮询规则改为插件内置

## 1.1.0 - 2026-07-23

- 战斗开始和结束默认发送 `_stop_all`
- 改为后台无窗口单文件 EXE
- 增加单实例保护，避免重复采集和重复事件
- 事件发送改为独立有界队列，避免网关请求阻塞遥测
- 限制 GameHub 地址为本机 `127.0.0.1`
- 配置刷新周期缩短到两秒
- 发布目录统一为 `artifacts`

## 1.0.0 - 2026-07-23

- 完成空战、陆战和 CAS 全部联动能力
- 接入 GameHub 离散事件和连续强度双通道
